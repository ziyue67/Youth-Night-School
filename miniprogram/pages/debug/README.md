# 云环境验证使用指南

## 如何在微信开发者工具中使用

### 方法1：通过小程序页面访问
1. 打开微信开发者工具
2. 在小程序中导航到：`pages/debug/env-test`
3. 点击页面上的"开始检查"按钮
4. 查看详细的环境诊断结果

### 方法2：通过控制台命令（不推荐）
该指令是为小程序页面设计的，不是终端命令。

## 具体使用步骤

### 第一步：部署云函数
1. 打开微信开发者工具
2. 点击左侧"云开发"图标
3. 进入"云函数"标签页
4. 确保所有云函数显示为"已部署"状态

### 第二步：测试环境
1. 在小程序中访问调试页面
2. 点击"开始检查"按钮
3. 查看每个云函数的状态

### 第三步：修复问题
根据页面显示的修复指南进行操作。

## 常见问题解决

### 云函数未部署
- 问题：显示"云函数未找到"
- 解决：在开发者工具中右键云函数 → 上传并部署

### 环境ID错误
- 问题：显示"云开发环境异常"
- 解决：确认云开发控制台中的环境ID是否为`cloud1-9go506hg40673425`

### 权限问题
- 问题：显示"权限配置异常"
- 解决：在云开发控制台设置数据库权限
